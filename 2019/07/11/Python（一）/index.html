Posted on 2019-07-11

安装pygame模块（pygame-1.9.2-cp35-cp35m-win_amd64.whl）时报错
输入

ERROR: Could not find a version that satisfies the requirement pygame-1.9.2-cp35-cp35m-win_amd64 (from versions: none)
ERROR: No matching distribution found for pygame-1.9.2-cp35-cp35m-win_amd64

我下载的pygame是一个whl文件
whl格式本质上是一个压缩包，里面包含了py文件，以及经过编译的pyd文件。使得可以在不具备编译环境的情况下，选择合适自己的python环境进行安装。

一开始我并不知道安装whl文件是要先安装wheel的

安装完后报错和之前一样
仔细比对指令后发现安装文件是要加上后缀名的

但依然报错
ERROR: pygame-1.9.2-cp35-cp35m-win_amd64.whl is not a supported wheel on this platform.

（ps:指令可以简写成pip install pygame-1.9.2-cp35-cp35m-win_amd64.whl）
这个错误是因为下载的whl文件的命名格式和系统支持的格式不同导致的

通过这两行代码可以查看自己本地pip支持安装的whl文件格式
